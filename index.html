Stream Commands
  • !destiny raids [username] [xbox/ps]
  • Shows raid completions for the specified user
  • !followage [username]
  • Shows the time the specified user has followed for
  • !uptime
  • Shows how long the stream has been up for
  • !sr [artist] - [songname]
  • Adds the specified song to the song queue
  • !rewards
  • Shows how many stream points you have. These are generally used for raffles and gambling with !raid
  • !raid [amount]
  • Gambles the desired amount of points in a minigame
  • !fight [username]
  • Fights the targeted player and has a chance to either win or lose rewards from or to the target
  • !steal
  • Steals a random amount of points from a random user
  • !roulette [amount] [red/green/black]
  • Bets the specified amount of points on the specified colour
  • !help
  • Brings up the link to this page, but if you're already here...?
  • !hype
  • Spams some hype emojis
  • !lurk
  • Tells everyone you've gone lurking
  • !what
  • What?
